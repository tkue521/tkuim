# TLMXB資管系(日)：LINUX作業系統
這是禮拜四 LINUX 作業系統的樹莓派練習，使用方法：

```bash
$ cd ~/Desktop
$ git clone https://github.com/tkue521/tkuim.git
```
載好後進到資料夾裡

```bash
$ cd tkuim/
```
此時用 `ls` 可以看到有三個示範

```bash
$ ls
01-基礎		02-個人辨識	03-團體辨識
```
